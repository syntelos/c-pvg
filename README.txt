page svg from text

  Generate "etc/page.svg" textboxes from preformatted plain text.

  Employ veritcal and horiontal whitespace.

  The first line has text class "ti".

  A line with a tab (HT '\t' 0x09) has a URL between the tab
  and the newline (LF '\n' 0x0A).  That line has class
  "ln", and is wrapped in a hypertext anchor.

  Text classes include "rm" for roman text, "tt" for
  typewriter text, "ti" for title, and "ln" for hypertext
  link.  As found in "etc/page.svg".

  References

    [TXT] https://datatracker.ietf.org/doc/html/rfc20
    [SVG] https://www.w3.org/TR/SVG
    [CSS] https://www.w3.org/TR/CSS
